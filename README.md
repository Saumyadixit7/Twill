Twill – Modern Fashion E-Commerce Website


Project Overview  ->Twill is a modern and fully responsive fashion e-commerce website built using React and Vite. The project is designed to deliver a seamless and engaging online shopping experience for users interested in clothing and fashion products.

The application showcases a clean UI, smooth animations, dark mode support, and a well-structured component architecture, making it an excellent frontend e-commerce project for learning and portfolio purposes.


Technology Stack  ->

Frontend Framework: React 19.2.0
Build Tool: Vite 7.2.4
Styling: Tailwind CSS 4.1.18
Icons: React Icons 5.5.0
Carousel: React Slick & Slick Carousel
Animations: AOS (Animate On Scroll) 2.3.4

Key Features ->

Modern Hero section with promotional banners and sale highlights
Product showcase for shirts and women’s fashion collections
Featured and top products section
Customer testimonials with carousel integration
Promotional banner sections
Newsletter subscription functionality
Dark/Light mode toggle support
Fully responsive and mobile-friendly design
Smooth scroll-based animations for enhanced user experience


Component Architecture ->

The project follows a clean and modular component structure:

Navbar – Navigation bar with dark mode toggle

Hero – Main promotional section

Products – Product listing display

TopProducts – Featured products section

Testimonials – Customer reviews carousel

Banner – Promotional banners

Subscribe – Newsletter subscription form

Footer – Website footer section

Popup – Notification popups


Assets ->

The project includes high-quality visual assets such as:
Hero promotional images

Decorative UI elements and patterns
